# What is Instapay?
 
Instapay offers a payment sdk which enables merchants to accept payments from their customers using just their fingerprint.

Any merchant is also able to onboard a user to the protocol, therefore creating a decentralized network of PoS machine
which act as both bank and atm for the unbanked.
