# Installing Drivers

## Platforms Supported

[Mantra MFS100](https://www.mantratec.com/products/Fingerprint-Sensors/MFS100-Fingerprint-Scanner)

The sdk has been tested on windows for now.
The Mantra gives a .net sdk which would be able to work on any device.

## Steps for windows

1. Install the Mantra Driver from Link below
 
 Mantra Driver are required for the system to detect the device.

 https://github.com/thearyanag/instapay/blob/master/MFS100Driver_9.2.0.0.exe

2. Install the Mantra Windows client from Link below

  Mantra Client is required for the system to interact with the device.
  It opens a local server on the system at port 3001 which can be used to interact with the device.

 https://github.com/thearyanag/instapay/blob/master/MFS100ClientService_9.0.3.8.exe
